# Produce ERP – Inventory & Order Management System

## Overview

'Produce ERP' is a backend system designed to manage **inventory, purchasing, suppliers, customers, and sales orders** for a produce distribution or wholesale business.

The system tracks inventory using **lot-based tracking with FIFO costing**, ensuring accurate inventory valuation and traceability.

Core capabilities include:

* Supplier management
* Purchase order tracking
* Lot-based inventory management
* Inventory transactions
* Customer sales orders
* Invoicing and payments
* Inventory adjustments

The system is designed around **auditability**, **traceability**, and **accurate cost tracking**.
